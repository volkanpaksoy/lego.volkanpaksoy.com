    • Set build: Police Helicopter Chase (60243)

    30 September 2020

    I like to small sets as much as I enjoy the larger ones. This one had a nice deal on Argos which is in all honesty the only reason I bought it. I bough 6 and built one of them. I’ll see what I’ll do with the rest in the future.

    I enjoyed the build process though.

    Also I started using the build instruction app rather than the paper booklet that came with the set. I find it a lot more convenient and useful than the paper version as you can rotate and zoom and see the model from all different angles.
