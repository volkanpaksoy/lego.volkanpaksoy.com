    • Organising Foliage

    06 December 2020

    I organized all that bulk LEGO but there is still a lot of work lying ahead: I also want to inventory every single element I own!

    I’m now going over the parts but this time I’m planning to go category by category as when I start building I want all related parts in one place as much as possible.

    I thought a good category to start would be foliage as I’m sure it’s not scattered around in my collection. It looks great in the large Wham drawer:

    Also looks good in BrickStock:

    Lessons Learned

    • Trying to identify every single piece is hugely time-consuming.
    • There is no way you can manage a whole inventory without using small bags. Drawers and boxes are great but I’ll definitely make use of bags more going forward.
